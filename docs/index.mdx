cnspec is an open source, cloud-native tool that evaluates the security of your entire infrastructure. Built upon Mondoo's advanced security data fabric, it's also a core component of the [Mondoo Platform](/start/what-is-mondoo), serving as both a CLI and an agent for scanning.

### What Is cnspec?

Get to know cnspec and [explore its capabilities](/cnspec/cnspec-about).

### Install cnspec

[Download and install cnspec](/cnspec/cnspec-adv-install/overview) on your assets.
